# Angular 19 Portfolio Website

A modern and responsive portfolio website built with Angular 19.

## Features

- Responsive Design
- Bootstrap 5
- SCSS
- Angular Routing
- Standalone Components
- About
- Skills
- Projects
- Experience
- Education
- Certifications
- Services
- Contact

## Technologies

- Angular 19
- TypeScript
- Bootstrap 5
- SCSS
- Font Awesome

## Installation

```bash
npm install
ng serve
```
